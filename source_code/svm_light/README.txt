3.	编写使用 SVM 工具的情感分类系统（支持向量机）
（1）	SVM功能：是监督学习（知道预测什么，即目标变量的分类信息 ）算法的一种，解决经典的二分类问题
（2）	相关概念：
利用分隔超平面将数据分隔开
点到分割面的距离为间隔，希望间隔最大
支持向量就是离分隔超平面最近的点，最大化支持向量到分隔面的距离
求解这个问题最大分隔面需要经过一系列转换：
求距离转为凸二次规划-》构造拉格朗日-》通过KKT条件和对对偶问题的求解-》

SMO（序列最小化）求出alpha 和b可以求出分隔超平面
SMO工作原理：每次循环中选择两个alpha进行优化处理，一旦找到一对合适的alpha，那么就增大其中一个，同时减小另一个。选择的alpha要满足在间隔边界之外的条件，而且还没有进行过区间化处理或者不再边界上。

利用核函数（Kernel） 的工具将数据转换成易于分类器理解的形状，将数据映射到高维空间
（3）	使用svm_light
svm_learn.exe train.txt train.model > dump1
利用训练模型 训练 训练集将结果放于dump1
训练集格式：【label index value】
+1 41:1 95:1 168:1 169:1 413:1 558:1
Label ：就是分类的种类
Index：是有顺序的索引，通常是连续的整数
Value：用来训练的数据，通常是实数


svm_classify.exe test.txt train.model result.output > dump2
利用训练集 测试 测试集 结果放在result.output （存放的是点到分隔面的距离）

（4）	利用svm_lignt进行情感分析
读取训练集中的单词，以列表形式存储极性和单词
提取训练集中不重复的词构成词表
将训练集和测试集转为svm_light需要的格式 label index value
调用svm_light对应的learn和classify程序完成训练和分类 调用外部命令cmd.bat
读取result.output距离 测试正确率，如果极性为+，距离>0  or极性- 距离<0就正确率+1
